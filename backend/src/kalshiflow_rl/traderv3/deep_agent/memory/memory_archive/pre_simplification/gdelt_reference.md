# GDELT CAMEO Taxonomy Reference

## CAMEO Event Codes

Events are coded using the CAMEO (Conflict and Mediation Event Observations) system:

### Cooperative Events
- **01-04**: Cooperative verbal (statements, appeals, consultations, intent)
- **05-08**: Cooperative material (diplomacy, aid, yielding)

### Neutral
- **09**: Investigate (inquiries, probes, audits)

### Conflictual Events
- **10-12**: Conflictual verbal (demands, disapproval, rejection)
- **13-14**: Conflictual actions (threats, protests)
- **15-20**: Conflictual material (force, reduced relations, coercion, assault, fighting)

## GoldsteinScale (-10 to +10)

Measures event impact on stability:
- **+7 to +10**: Major cooperation (treaties, aid)
- **+1 to +6**: Mild cooperation (statements, consultations)
- **-1 to -6**: Mild conflict (demands, disapproval, threats)
- **-7 to -10**: Major conflict (force, assault, fighting)

## QuadClass (4 categories)

1. Verbal Cooperation
2. Material Cooperation
3. Verbal Conflict
4. Material Conflict
